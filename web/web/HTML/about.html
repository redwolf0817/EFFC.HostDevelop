jquery tmpl 详解

1.引入脚本

代码如下:

<script src="~/Scripts/jquery.tmpl.js"></script>

2.编写模板

后台json格式发送来的数据:
[{"tId":1,"tName":"张三","tAge":14,"tClass":1,"TeachClass":{"cId":1,"cName":"1班"}},
模板代码如下:
<script type="text/x-jquery-tmpl" id="testTemplate">
<tr>
  <td>${tid} </td>
  <td>${tName} </td>
  <td>${tAge} </td>
  <td>${teachClass.cName} </td>
</tr>
</script>

3.填写模板

代码如下:
<script >
$(function(){
$.getJson("/Home/List",null,function(jsonData){
$("#testTemplate").tmpl(jsonData).appendTo("#List");
  });
});
</script>
